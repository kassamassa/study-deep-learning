# study-deep-learning
Implementation of Deep Learning fundamentals **from scratch** using **NumPy**.

# Project Overview
This repository is dedicated to understanding the internal mechanisms of Neural Networks by
implementing them from the ground up. By avoiding high-level frameworks like PyTorch or
TensorFlow and relying solely on NumPy, I aim to grasp the mathematical foundations and
architectural logic of Deep Learning.

# Goals
**Deep Understanding:** Mastering backpropagation, optimization algorithms, and layer architectures through manual implementation.
**Preparation for E-Certification:** Solidifying the theoretical knowledge required for the **JDLA E-Certification** (August 2026).
* **Skill Enhancement:** Building robust **hands-on implementation skills** to bridge the gap between theory and real-world application.
